GitHub Nudge Bot Privacy Policy

This privacy policy describes how GitHub (“we”, “us”, “our”) collects and uses the information you provide when using our pull request application (the “Nudge Bot”). This Application is hosted on GitHub and is open source.

What information do we collect?

We do not collect any personal data from you or from any third party services. The only data we store is the data necessary to expedite the pull request approval process. This data includes any pull requests that have been created, any comments made on the pull requests, and any changes made to the pull request.

How do we use your information?

We use the data we store in order to expedite the pull request approval process. We do not share any of this data with any third party services.

How do we store your information?

We store your data securely on our servers. We make sure to use the latest security protocols and encryption techniques to keep your data safe.

How can you access, update, or delete your information?

Simply uninstalling the application will completely remove the all the user and the pull requests data. But if you would still like to connect and access, update, or delete any of the data we have stored about you, you can contact us at suhailgupt03@gmail.com

Changes to this privacy policy

We may make changes to this privacy policy from time to time in order to keep up with changes in the law or our Application. We will notify you of any changes by posting the updated policy on our website.
